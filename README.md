# Social Media Application

This is a basic **console-based Social Media Application** developed using **Java** following the **MVC (Model-View-Controller)** architecture.

## Features

- User Login
- Add Users
- Send Friend Requests
- Accept Friend Requests
- Create Posts
- Add Comments
- Notifications
- View User Suggestions

## Technologies Used

- Java
- OOP Concepts
- Collections Framework
- MVC Design Pattern

## Project Type

This project is designed for learning purposes and demonstrates how a social media platform can work in a simple command-line environment.

## How to Run

1. Open project in VS Code / Eclipse / IntelliJ  
2. Compile Java files  
3. Run `Main.java`
